# SQL Data Analysis Project

## Overview
Built a relational database to analyze authentication activity and system performance.

## Results
- 210+ login records analyzed
- Reduced failure rate from 30.5% → 17.1%
- Improved success rate to 82.9%

## Key Analysis
- Login success vs failure
- Failed login patterns by user
- Ticket distribution by department
